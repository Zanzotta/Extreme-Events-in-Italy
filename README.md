# Extreme Events in Italy Analysis

**Streamlit App:** [Live Demo](https://extreme-events-in-italy-hxcxfacpdkbcorhyjwl5by.streamlit.app/)

## Overview
This app analyzes the frequency and trends of extreme events in Italy since the 19th century, using the EM-DAT emergency events dataset. The goal is to visualize whether extreme events (like floods, earthquakes, and storms) are becoming more frequent over time.

## Features
- **Bar Chart:** Shows the most frequent types of extreme events in Italy.
- **Heatmap:** Visualizes the frequency of extreme events over time, highlighting trends.
- **Interactive Filters:** Allows users to select specific types of extreme events for analysis.

## Dataset
The dataset used is the [EM-DAT Emergency Events Dataset](https://dataverse.uclouvain.be/dataset.xhtml?persistentId=doi:10.14428/DVN/I0LTPH), which records extreme events globally. This app focuses on events in Italy.
